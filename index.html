C/C++ and Python Scripts for Optics and Electromagnetics

Computes the power reflected and transmitted from an interface between two media using Fresnel equations..

Here is a list of public repositories currently available on my GitHub page :

FresnelCoefficients

Computes the power reflected and transmitted from an interface between two media using Fresnel equations..

TMM

Computes the power reflected and transmitted from a stack of dielectric or metallic layers using the transfer matrix method.

MieScattering

Computes the electromagnetic fields scattered by a dielectric or metallic sphere using Mie theory.

MieScatteringPEC

Computes the electromagnetic fields scattered by a PEC sphere using Mie theory.
